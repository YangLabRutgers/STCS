# STCS: Spatial Transceiptomics Cell Segmentation

Here is STCS, a pipeline that reconstruct subcellular barcodes from sequence-based Spatial Transceiptomics data into single cell level segmented data.

More details in the [tutorial notebook](https://github.com/XH300/SpatialMethod/blob/Sun-modified/STCS_tutorial.ipynb)

To run the tutorial with the same ground truth data we used, sequencing data can be downloaded from [Visium HD Spatial Gene Expression Libraries, Post-Xenium, Human Lung Cancer (FFPE)](https://www.10xgenomics.com/datasets/visium-hd-cytassist-gene-expression-human-lung-cancer-post-xenium-expt) Corresponding tif image can be downloaded from [Post-Xenium Technical Note: Xenium v1 and Xenium Prime 5K for FFPE Human Lung Cancer](https://www.10xgenomics.com/datasets/xenium-human-lung-cancer-post-xenium-technote)
The data we used was from Experiment 2, post-Xenium Prime 5K.
